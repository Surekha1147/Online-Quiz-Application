# Online Quiz Application

## Overview

The Online Quiz Application is a web-based interactive platform developed using HTML, CSS, and JavaScript. It is designed to provide users with an efficient and engaging environment for answering multiple-choice questions, tracking scores, and improving knowledge through timed assessments. The application operates entirely on the client side and delivers a smooth user experience with responsive design and modern interface elements. :contentReference[oaicite:0]{index=0}

---

## Objectives

- Develop a dynamic and responsive quiz platform  
- Implement random question selection for varied attempts  
- Provide instant feedback and real-time score updates  
- Improve user experience through clean interface design  
- Enhance learning through time-based assessments :contentReference[oaicite:1]{index=1}

---

## Key Features

- Multiple-choice questions with four answer options  
- Timer-based quiz functionality  
- Automatic skip when time expires  
- Real-time score calculation  
- Progress tracking through progress bar  
- Dark mode toggle for better accessibility  
- Exit option to end quiz session anytime  
- Responsive layout compatible with multiple devices :contentReference[oaicite:2]{index=2}

---

## Technology Stack

- **HTML5** – Defines the structure and content of the application  
- **CSS3** – Provides styling, animations, and responsive layout  
- **JavaScript** – Handles quiz logic, timers, scoring, and DOM manipulation  
- **Local Storage** – Stores user high scores locally in the browser :contentReference[oaicite:3]{index=3}
